Udemy Clone

A simple and responsive Udemy landing page clone built using pure HTML and CSS.
This project recreates the opening page design of the popular online learning platform Udemy.

🚀 Features
Responsive navigation bar
Search box UI
Trending courses section
Recommended courses cards
Popular topics section
Most popular courses section
Attractive footer design
Clean and modern UI
🛠️ Technologies Used
HTML5
CSS3
Font Awesome Icons
